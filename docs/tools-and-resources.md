# 工具与资源

## 常用开发工具

- **VS Code**：主流的跨平台代码编辑器，丰富的插件生态。
- **PyCharm/IntelliJ IDEA**：专业的 Python/全栈开发 IDE。
- **Postman**：API 测试与调试工具。
- **Docker Desktop**：容器管理与镜像构建。
- **DBeaver/pgAdmin**：数据库可视化管理工具。
- **Git**：版本控制，推荐配合 GitHub/GitLab。
- **Zsh/Oh My Zsh**：高效终端与命令行增强。
